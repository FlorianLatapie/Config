## Raccourcis clavier
- Générer le diagramme UML du projet : `Ctrl + Alt + Shift + U`
- Reformater le fichier courant : `Ctrl + Alt + Shift + L` (Cochez aussi la case "Optimize imports")
- Renommer une variable : `Shift + F6`
